# Tic-Tac-Toe

A browser-based Tic Tac Toe game built using **vanilla JavaScript**, HTML, and CSS.

This project was created to practice and understand:

- Factory Functions
- Module Pattern (IIFE)
- Closures
- DOM Manipulation
- Event Listeners
- Game State Management
- Rendering Logic
- Application Flow Architecture

---

# ✨ Features

- Two-player Tic Tac Toe
- Dynamic player names using form input
- Win detection
- Draw detection
- Invalid move highlighting
- Overlay announcements for:
  - Winner
  - Draw

- Reset button
- Automatic board rendering
- Clean modular JavaScript structure

# 📚 What I Learned

This project helped me understand the difference between:

- Logic vs UI rendering
- Game state vs display state
- Factory functions vs IIFEs
- Single round flow vs full application lifecycle

It also improved my understanding of organizing JavaScript into reusable modules instead of writing everything globally.
